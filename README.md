# 💳 Java ATM System (Console App)

A simple Java console-based ATM application.

## 🚀 Features

- User login (with PIN)
- Deposit and withdraw
- Balance check
- Mini-statement (transaction history)
- Change PIN
- Multiple accounts

## 🧪 How to Run

1. Clone this repository or download the files
2. Compile the code:

```bash
javac ATM_System.java
java ATM_System
